Your copy-trading EA ▸
The AntiGreed bot runs on the admin's account. To mirror its trades onto your MT5 account, install the AntiGreedCopier Expert Advisor below and paste your credentials. Every time the bot opens or closes a trade, the EA replays it on your terminal.
Rotating the key invalidates the one in your installed EA — paste the new value into the EA after rotating.
1
Download AntiGreedCopier
Download the bundle, unzip, and merge into MT5's data folder under MQL5\: the Experts folder contents go in MQL5\Experts\, the Files folder contents in MQL5\Files\ (these are the icons the chart panel renders).
Easiest way to find the folder: in MT5 click File → Open Data Folder, then open MQL5.
MT5 has no native Mac build — download the Mac wrapper from your broker (Exness: exness.com/downloads/mt5/macos) and install it. Then download the bundle below, unzip, and merge into the data folder under MQL5/ — Experts/ contents into MQL5/Experts/, Files/ contents into MQL5/Files/.
In MT5 click File → Open Data Folder — it opens the right path inside the Wine prefix (somewhere under ~/Library/Application Support/…/drive_c/…). On Apple Silicon expect a small first-launch delay while Rosetta translates the .exe.

2
Compile in MetaEditor
Open MT5 → press F4 → open AntiGreedCopier.mq5 → press F7. Should report "0 errors, 0 warnings".
Open MT5 → press F4 (or Tools → MetaQuotes Language Editor) → open AntiGreedCopier.mq5 → press F7. Should report "0 errors, 0 warnings".
If F4 doesn't fire under macOS, check System Settings → Keyboard — function keys may be assigned to media controls; hold fn with F4/F7, or untick "Use F1, F2, etc. keys as standard function keys".
3
Whitelist the API URL
In MT5: Tools → Options → Expert Advisors → tick "Allow WebRequest for listed URL" and add the API base URL above. Otherwise MT5 blocks outbound HTTP.
In MT5: MetaTrader 5 → Settings → Expert Advisors (or Tools → Options in older builds) → tick "Allow WebRequest for listed URL" and add the API base URL above. Otherwise MT5 blocks outbound HTTP and you'll see WebRequest error 4014 in the journal.
4
Attach to any chart
Drag the EA from Navigator onto any chart (symbol doesn't matter — the EA picks symbols from the signals). In the Inputs tab, paste the API URL, the API key, and your AD-ID from above.
Optional: leave EnabledSymbols empty to copy every trade the admin opens, or list specific pairs (e.g. EURUSD,GBPUSD,XAUUSD) to only copy those. The on-chart panel will show the active whitelist.
In the Navigator panel (⌘+N to toggle), expand Expert Advisors, then drag AntiGreedCopier onto any chart (the symbol doesn't matter — the EA picks symbols from the signals). In the Inputs tab, paste the API URL, the API key, and your AD-ID from above.
If you don't see the EA in the Navigator after compiling, right-click Expert Advisors → Refresh. Optional: EnabledSymbols lets you filter — leave empty to copy every trade, or list pairs (e.g. EURUSD,GBPUSD,XAUUSD).
5
Enable Algo Trading
Click the green Algo Trading button in the MT5 toolbar. The EA's smiley face should appear in the top-right of the chart. You're done.
Click the green Algo Trading button in the MT5 toolbar (or ⌘+E). The EA's smiley face should appear in the top-right of the chart. You're done.
If MT5 quits or sleeps, the EA stops copying. Either disable App Nap on the MT5 app (right-click MT5 in Finder → Get Info → tick "Prevent App Nap" if shown) or keep the laptop awake while trades are open — most operators just run MT5 on an always-on cheap VPS.
